# Template-CED-Internal-Repo
Internal documentation to support timely updates of data products and team redundancies.


If you are just getting started with data work at the Atlanta Fed, please see [the SharePoint onboarding folder](https://frbprod1.sharepoint.com/:f:/r/sites/6F-CED/Shared%20Documents/DATA%20Working%20Group/2026/Priority%201%20-%20Redundancies/Data%20Product%20Onboarding?csf=1&web=1&e=tcFTOz).

This repo has four folders.
  - src: for source code
  - doc: for documentation, like a data dictionary, methodology, data product user manual, and contacts
  - dep: where dependencies are stored
  - test: for code to validate output

README.md structure:
  - Project Name
  - Brief Project Description
  - Directions to critical files
  - Any intricacies that the redundancy should be aware of
